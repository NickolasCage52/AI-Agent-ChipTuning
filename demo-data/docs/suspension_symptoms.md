# Симптомы подвески (демо-документ)

## section: knock_turning

- Стук при повороте может быть связан с: стойкой стабилизатора, шаровой опорой, рулевым наконечником, опорным подшипником.
- Достоверный вывод делается только после осмотра (подъёмник/проверка люфтов).

## section: safety_note

- Тормоза/рулевое/подвеска — безопасность. При выраженном стуке лучше не откладывать диагностику.

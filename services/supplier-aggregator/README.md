## supplier-aggregator (MVP)

В MVP логика импорта/нормализации прайсов реализована в `core-api` (`app/supplier_import.py`).

Этот модуль оставлен как отдельная зона ответственности для будущего вынесения:
- нормализация прайсов разных поставщиков
- обогащение (кроссы/OEM)
- ранжирование по цене/наличию/сроку
